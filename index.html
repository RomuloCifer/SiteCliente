Vocabulário Essencial para Reuniões

"Let's touch base on..."

"Let's touch base on the project status next week."

Significado: "Vamos conversar/alinhar sobre..."
Uso: Para sugerir uma breve reunião ou atualização sobre um assunto.

"Moving forward..."

"Moving forward, we'll implement these new procedures."

Significado: "Daqui para frente..." ou "A partir de agora..."
Uso: Para indicar mudanças ou planos futuros.

"On the same page"

"Are we all on the same page regarding the deadline?"

Significado: "Estar em sintonia/acordo"
Uso: Para confirmar que todos têm o mesmo entendimento.

"Bottom line"

"The bottom line is we need to increase sales."

Significado: "Em resumo" ou "O mais importante"
Uso: Para enfatizar o ponto mais importante ou a conclusão.

"Win-win situation"

"This partnership would be a win-win situation for both companies."

Significado: "Situação benéfica para ambos os lados"
Uso: Para descrever um resultado positivo para todas as partes.

"To follow up"

"I'll follow up with you after the meeting."

Significado: "Dar continuidade" ou "Acompanhar"
Uso: Para indicar que você dará sequência a um assunto.

"To keep in the loop"

"Please keep me in the loop on any developments."

Significado: "Manter informado"
Uso: Para pedir ou prometer manter alguém atualizado.

"To run through"

"Let's run through the main points quickly."

Significado: "Revisar rapidamente"
Uso: Para sugerir uma revisão rápida dos pontos principais.

"To wrap up"

"Let's wrap up this meeting in the next five minutes."

Significado: "Concluir" ou "Finalizar"
Uso: Para indicar que está chegando ao fim de algo.

"To take it offline"

"We can take this discussion offline."

Significado: "Continuar a discussão em outro momento/local"
Uso: Para sugerir continuar uma discussão específica fora da reunião atual.
